3.4 Documentos Largos (20K+ tokens)

📋 Estrategia Recomendada por Anthropic
  1. Documentos ARRIBA — Contenido largo al principio
  2. Query al FINAL — Instrucciones al final (+30% calidad)
  3. Estructura XML — <document index="n">
  4. Cita primero — Extraer quotes relevantes antes de analizar
  5. Metadatos — fuente, fecha, tipo en atributos XML
💡 DeepSeek: Con 128K de contexto, usa la misma estrategia. Para documentos que excedan el contexto, considera RAG (Retrieval Augmented Generation).
📋 Strategy Recommended by Anthropic
  1. Documents UP TOP — Long content at the start
  2. Query at the END — Instructions last (+30% quality)
  3. XML structure — <document index="n">
  4. Quote first — Extract relevant quotes before analyzing
  5. Metadata — source, date, type as XML attributes
💡 DeepSeek: With 128K of context, use the same strategy. For documents that exceed the context window, consider RAG (Retrieval Augmented Generation).

5.2 Adaptive Thinking

Claude Opus 4.7 decide dinámicamente cuándo pensar, basado en complejidad + effort.

# Si piensa DEMASIADO (sube latencia sin justificarlo):
"Pensar añade latencia. Úsalo solo cuando mejore
significativamente la calidad de la respuesta. Si dudas, responde directamente."

# Si piensa POCO (sube primero el effort, luego refuerza por prompt):
"Tras recibir resultados de herramientas, reflexiona con cuidado
sobre su calidad y decide los próximos pasos óptimos antes de continuar."
💡 Migración: el antiguo thinking={"type":"enabled","budget_tokens":32000} se reemplaza por thinking={"type":"adaptive"} + output_config={"effort":"xhigh"}.

Claude Opus 4.7 dynamically decides when to think, based on complexity + effort.

# If it thinks TOO MUCH:
"Thinking adds latency. Only use it when it meaningfully
improves answer quality. When in doubt, respond directly."

# If it thinks TOO LITTLE (raise effort first, then prompt):
"After receiving tool results, carefully reflect on their
quality and determine optimal next steps before proceeding."
💡 Migration: The old thinking={"type":"enabled","budget_tokens":32000} is replaced by thinking={"type":"adaptive"} + output_config={"effort":"xhigh"}.

🖼️ Técnicas para Vision Prompts
  • Sé espacialmente específico: "En la esquina superior derecha..."
  • Describe el contexto esperado: "Esto es un recibo médico de 2024"
  • Pide coordenadas: "Lista los elementos y su posición (x, y)"
  • PDFs = imágenes: Claude ve PDFs página por página como imágenes
  • Multimodal Chain of Thought: "Describe la imagen → analiza → responde"
