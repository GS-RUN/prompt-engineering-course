🪄 SKILL.md Builder

🎯 Agent Goal Composer

Estructura un prompt de agente con rol, objetivo, restricciones y criterios de éxito

El prompt sale envuelto en tags XML (Claude / Anthropic-friendly). Para modelos pequeños abiertos puedes simplificar a Markdown plano.
The Agent Goal Composer runs in Spanish mode. Switch to ES to compose a well-structured agent prompt.

🩺 Prompt Linter

Pega un system prompt y recibe checks sobre rol, ejemplos, formato y longitud

Checks rápidos basados en la guía de prompt engineering de Anthropic: rol, ejemplos, formato, longitud, XML, CoT.
The Prompt Linter runs in Spanish mode. Switch to ES to lint a system prompt against Anthropic-style best practices.
