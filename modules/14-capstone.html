🚧 Próximamente

Los proyectos capstone son la pieza más demandante del curso y están en desarrollo activo. Cada uno incluirá: starter repo con esqueleto reproducible, especificación detallada con criterios de aceptación, solución de referencia y rúbrica de evaluación para auto-evaluarte.

Proyecto A — Code Review Bot end-to-end

Bot que revisa Pull Requests automáticamente: integración con GitHub Actions, system prompt robusto, structured outputs, JSON Schema, prompt caching, evals automatizados con golden dataset de 50 PRs reales. Stack: Python + Anthropic API + GitHub API.

Proyecto B — RAG Knowledge Assistant

Asistente que responde sobre tu base de conocimiento (Confluence, Notion, Google Drive, mil PDFs). Pipeline RAG completo: ingesta + chunking + embeddings + vector DB + retrieval + augmented generation + citation. Stack: Python + LangChain + Qdrant + Cohere/OpenAI embeddings + Claude.

Proyecto C — Local Self-Hosted Assistant

Asistente 100% local con privacidad total: Qwen3-32B Q5 vía Ollama + RAG con ChromaDB + frontend Next.js. Métricas: latencia, calidad, cumplimiento GDPR. Hardware target: RTX 4090 / 5090 o Mac M3/M4 Pro.

Proyecto D — Multi-Agent Research Team

Equipo de agentes que investiga un tema y escribe un informe: researcher (busca en web), analyst (sintetiza), critic (red-teams), writer (redacta). Patrón supervisor-worker con LangGraph + Claude Code skills + MCP tools. Métricas: cobertura, accuracy factual vs golden answers, coste total.

Cuándo: primer proyecto disponible Q3 2026. Si quieres priorizar uno, abre issue en GitHub con la propuesta.
